CacheMap

CacheMap

Class for keeping track of a mapping from keys to values, creating new entries using a given function when a key that hasn't yet been recorded is requested.

Constructor

new CacheMap(createEntryFunc)

Parameters:
Name Type Description
createEntryFunc function

Function for creating an entry.

Methods

get(key)

Get an entry in the map. If there is no such entry, the createEntryFunc parameter passed in the constructor is called, and its result is recorded in the map and then returned.

Parameters:
Name Type Description
key any

Key to use for referring to the entry.
